# CLI Calculator

A command-line calculator built with Node.js as part of my backend learning journey.

## Run

```bash
node calculator.js
```

## Features

- Addition, Subtraction, Multiplication, Division an module
- Division by zero error handling
- Invalid input detection
- Repeat calculation without restarting

## What I learned

- JavaScript functions
- Error handling (try/catch)
- Node.js readline module
