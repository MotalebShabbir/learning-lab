## 🧱 Why This Order?

Because fundamentals without application are trivia, and application without fundamentals is a house of cards.

- **CS first** — so every line of code I write has a reason.
- **Backend heavy** — because the backend is where trust, data, and logic live.
- **Full Stack** — to understand the complete picture before specializing.
- **AI Integration** — to wield the most powerful tool of this era, not be replaced by it.
- **Cyber Security last** — because you cannot defend what you do not fully understand.
